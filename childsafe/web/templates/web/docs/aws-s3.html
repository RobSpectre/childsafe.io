Setup childsafe.io on AWS S3

Run this Cloud Formation Template to make your site childsafe in less than three minutes. Watch this video to see the whole process from beginning to end.

Paste your childsafe.io ID

We're going to set up a lamda function that will send a POST request to childsafe.io whenever a new image is uploaded to your S3 bucket. Included in that post request will be a image URL, a resource ID, and your childsafe.io user ID.

To get started, copy your childsafe.io ID from your user profile. Then run this Cloud Formation Template, click Next, and then paste your childsafe.io ID into the text box. Then click Next again.

Create a lamda function

No further customization is needed after you've added your childsafe ID. Just click next, acknowledge the terms and conditions, create your platform, and execute. You may need to refresh a couple times while waiting for your function to be created.

Set an S3 Trigger

Once your function is created, click *Triggers* and select S3 as the triggering integration. Choose the bucket user generated content is stored in, and select *Object Created (All)* as the *Event Type*. Click next.

Your site is now childsafe.
